# 正则表达式(Regular Expression)



​	

## 字段

| [ a-z ] |         |
| ------- | ------- |
| { n,m}  |         |
| \w      | 匹配a-z |
| \d      | 匹配0-9 |
|         |         |
|         |         |
|         |         |
|         |         |

## 范例

```ASN.1
a{6,10}		// a 出现 6-10 次
b{,10}		// b 出现 0-10 次
c{5,}		// c 出现 5-& 次
\w{5,10} 	// 单词字符出现 5-10 次
```
